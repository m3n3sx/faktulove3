Keyboard Shortcuts Reference

Quick reference for all keyboard shortcuts in FaktuLove

Globalne skróty

Skrót Opis
Ctrl+n Utwórz nową fakturę
Ctrl+Shift+n Dodaj nowego kontrahenta
Ctrl+u Przejdź do OCR
Ctrl+h Przejdź do panelu głównego
Ctrl+/ Pokaż pomoc
esc Zamknij modal

Formularz faktury

Skrót Opis
Ctrl+s Zapisz fakturę
Ctrl+Shift+s Zapisz i utwórz nową
Ctrl+d Duplikuj fakturę
Alt+a Dodaj pozycję

Lista faktur

Skrót Opis
Ctrl+f Szukaj faktur
Ctrl+e Eksportuj faktury

Wskazówki

  • Skróty globalne działają na każdej stronie aplikacji
  • Skróty kontekstowe działają tylko na odpowiednich stronach
  • Użyj Ctrl+/ aby wyświetlić pomoc ze skrótami w aplikacji
  • Wszystkie skróty są dostępne dla użytkowników klawiatury
